Schroeder-Bernstein theorem, proof of


We first prove as a lemma that for any B⊂A, if there is an injection f:A→B, then there is also a bijection h:A→B.

Inductively define a sequence (Cn) of subsets of A by C0=A∖B and Cn+1=f⁢(Cn). Now let C=⋃k=0∞Ck, and define h:A→B by

h⁢(z)={f⁢(z),z∈Cz,z∉C.

If z∈C, then h⁢(z)=f⁢(z)∈B. But if z∉C, then z∈B, and so h⁢(z)∈B. Hence h is well-defined; h is injective by construction. Let b∈B. If b∉C, then h⁢(b)=b. Otherwise, b∈Ck=f⁢(Ck-1) for some k>0, and so there is some a∈Ck-1 such that h⁢(a)=f⁢(a)=b. Thus h is bijective; in particular, if B=A, then h is simply the identity map on A.

To prove the theorem, suppose f:S→T and g:T→S are injective. Then the composition g⁢f:S→g⁢(T) is also injective. By the lemma, there is a bijection h′:S→g⁢(T). The injectivity of g implies that g-1:g⁢(T)→T exists and is bijective. Define h:S→T by h⁢(z)=g-1⁢h′⁢(z); this map is a bijection, and so S and T have the same cardinality.
